[h1]Fallout Immersion Project - Future-Tec[/h1]
[i]"Vault-Tec planned for every future - especially the bad ones."[/i]

[url=https://ko-fi.com/misterhouse][img]https://i.imgur.com/YUpX4Up.png[/img][/url]

[b][Open Beta][/b] This mod is under active development - updates and patches may drop anytime.
Your feedback directly shapes development!

[h2]Features[/h2]
[list]
[*] [b]Vanilla Quests Expanded - Ancients:[/b] rewrites ancient complexes, discoveries, experiments, and quest language as [b]Vault-Tec blacksites[/b], FEV research, and sealed pre-war programs.
[*] [b]Vanilla Quests Expanded - Cryptoforge:[/b] converts the quest chain and its central facility into a [b]mobile base crawler[/b] and compatible pre-war technology narrative.
[*] [b]Vanilla Quests Expanded - Deadlife:[/b] reframes its sites and events as [b]continuity bunkers[/b], [b]feral-ghoul[/b] outbreaks, and [b]Operation Deadlife[/b].
[*] [b]Vanilla Quests Expanded - Generator:[/b] converts the story to [b]Lee Moldaver[/b], a restoration reactor, and fusion-core terminology, including the supported character's name and gender presentation.
[*] [b]Full quest-language pass:[/b] updates letters, objectives, descriptions, rewards, incidents, site labels, and related terminology for each supported quest module.
[*] [b]Mechanics scope:[/b] objectives, maps, encounters, rewards, triggers, and quest logic remain those of the original Vanilla Quests Expanded modules.
[*] [b]Visual scope:[/b] Future-Tec currently contains no in-game texture replacements; it is a narrative and localization compatibility module.
[/list]

[h2]Requirements[/h2]
[list]
[*] No DLC or external mod is a hard requirement.
[/list]

[h2]Recommended[/h2]
[list]
[*] [b]Vanilla Quests Expanded - Ancients[/b] - Optional Vault-Tec blacksite and FEV conversion.
[*] [b]Vanilla Quests Expanded - Cryptoforge[/b] - Optional mobile base-crawler conversion.
[*] [b]Vanilla Quests Expanded - Deadlife[/b] - Optional continuity-bunker and feral-ghoul conversion.
[*] [b]Vanilla Quests Expanded - Generator[/b] - Optional Lee Moldaver restoration-reactor conversion.
[/list]

[h2]Development Notes[/h2]
[list]
[*] Each quest integration is conditional and can be used independently; Future-Tec does not require the full Vanilla Quests Expanded collection.
[*] The module does not add replacement quest logic. It changes the fiction, terminology, and the defined Generator character presentation around the source quests.
[*] Quests already generated in a save can retain cached text; newly generated quests receive the full conversion.
[*] [url=https://github.com/BavarianBarbarian21/Fallout-Immersion-Project]Source code and development repository[/url]
[*] This mod is part of the Fallout Immersion Project (FIP) - a modular overhaul inspired by Fallout's lore. FIP is not affiliated with the Fallout Collaboration Project (FCP), but is designed to complement it. [url=https://discordapp.com/channels/644497305650593793/1414315188483657869]FIP channel in the FCP Discord[/url]
[/list]
