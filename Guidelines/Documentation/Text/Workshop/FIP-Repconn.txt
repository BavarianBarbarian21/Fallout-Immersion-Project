[b][Open Beta][/b] This mod is under active development - updates and patches may drop anytime.
Your feedback directly shapes development!

[h2]Features[/h2]
[list]
[*] Rebuilds Fallout [b]aerospace[/b] gameplay around [b]Odyssey gravships[/b] instead of leaving ship content in a vanilla frame.
[*] Adds dedicated Odyssey patching and language overrides so gravship systems feel like part of the Fallout setting.
[*] Includes compatibility support for [b]Vanilla Gravship Expanded[/b].
[*] Keeps gravship and aerospace integration isolated in one module rather than spreading ship patches across unrelated FIP content.
[/list]

[h2]Requirements[/h2]
[list]
[*] [b]Odyssey DLC[/b] - Required (core gravship functionality)
[/list]

[h2]Recommended[/h2]
[list]
[*] [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3609835606][b]Vanilla Gravship Expanded - Chapter 1[/b][/url] - Optional (extra compatibility patch coverage)
[/list]

[h2]Development Notes[/h2]
[list]
[*] This module is in open beta. Balance passes, naming cleanup, and compatibility refinements are still ongoing, so targeted feedback is useful.
[*] This mod is part of the Fallout Immersion Project (FIP) - a complete modular overhaul inspired by Fallout's lore. FIP is not affiliated with the Fallout Collaboration Project (FCP), but it is designed to complement FCP and other Fallout-friendly RimWorld mods. You can also find the project in the FCP Discord's FIP channel: [url=https://discordapp.com/channels/644497305650593793/1414315188483657869]Join here[/url]
[/list]
